# eslint-config

[![npm][npm-image]][npm-url]
[![build][build-image]][build-url]
[![license][license-image]][license-url]

Personal [eslint](https://github.com/eslint/eslint) config.

## Install

`$ npm i @akijoey/eslint-config -D`

## Usage

Add to your eslint config:

```json
{
  "extends": "@akijoey"
}
```
